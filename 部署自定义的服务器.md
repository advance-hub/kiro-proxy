# FRP 服务器部署（Linux 一键）

## 部署


```bash
# 1. 下载
cd /opt && wget -qO- https://github.com/fatedier/frp/releases/download/v0.67.0/frp_0.67.0_linux_amd64.tar.gz | tar xz
cd frp_0.67.0_linux_amd64

# 2. 写配置（改 TOKEN 为你自己的密码）
cat > frps.toml << 'EOF'
bindPort = 7000
auth.token = "你的密码"
vhostHTTPPort = 8080
EOF

# 3. 启动
./frps -c frps.toml &
```

## 开放端口

云控制台安全组放行：
- **7000**（FRP 通信）
- **8080**（HTTP 穿透）

TCP 模式还需放行你配置的远程端口。

## 填入 Kiro Launcher

穿透页面填写：

| 字段 | 值 |
|------|------|
| 服务器地址 | 你的服务器 IP |
| 服务器端口 | 7000 |
| 认证 Token | 你设的密码 |
| HTTP 端口 | 8080 |

## 域名（可选）

DNS 添加 A 记录 `kiro.example.com → 服务器IP`，然后在 Launcher 填自定义域名即可。

访问地址：`http://kiro.example.com:8080/v1`

## 常用命令

```bash
systemctl status frps     # 状态
systemctl restart frps    # 重启
journalctl -u frps -f     # 日志
```

## 卸载

```bash
systemctl stop frps && systemctl disable frps
rm -rf /opt/frp_0.67.0_linux_amd64 /etc/systemd/system/frps.service
systemctl daemon-reload
```
